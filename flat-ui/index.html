Flat UI 演示系统

页面 (Pages)

登录页面

用户登录界面，包含账号、密码和验证码输入

用户管理

用户列表管理，支持添加、编辑、删除用户操作

站点管理

设备管理和地图展示，媒体流播放功能

弹窗 (Modals)

添加用户

新增用户弹窗，包含角色选择和密码生成

编辑用户

修改用户信息弹窗，支持角色和地区设置

修改密码

密码修改弹窗，包含强度检测和生成功能

设备概览

设备列表查看弹窗，支持分页和筛选

角色权限管理

角色权限矩阵管理，可配置不同角色的权限
说明： 此 flat-ui 系统是从原有模块系统中抽取的独立版本。每个页面包含内联 CSS 样式，JS 文件负责业务逻辑和事件绑定。所有功能均为模拟实现，实际使用时需要连接真实的 API 接口。原有的模块系统保持完全不变。
